Justin Dorfman

Open Source Sustainer

Justin Dorfman is StackShare's Developer Evangelist and is responsible for championing the needs of the open source software community. Previously, Justin led Developer Relations as well as Open Source initiatives for Sticker Mule & MaxCDN.

Justin is heavily involved with the FOSS community contributing to Bootstrap, Font Awesome, Grunt, jQuery, Twemoji, Nginx, GNU Bash, and many more. He also serves on the Selection Committee for Mozilla's Open Source Support (MOSS) program, as well as the Open Source Collective's board. He is also a co-founder/organizer of Sustain Summit, a one-day event for Open Source Software Sustainers.

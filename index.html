NAME

Justin Dorfman - Open Source Sustainer, Maintainer, & Contributor

SYNOPSIS

Justin Dorfman is Sourcegraph's Open Source Community Manager and is responsible for fostering the adoption of code search and context-aware AI in the open source community.

Justin has contributed to Bootstrap, Font Awesome, jQuery, Nginx, CNCF, GNU Bash, Zsh, and many more. He also served on the selection committee for Mozilla's Open Source Support (MOSS) program. In 2017, he co-founded SustainOSS, which hosts events and podcasts for Open Source Software Sustainers. Additionally, he is a board member at the Open Source Collective.
